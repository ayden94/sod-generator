time input의 시간 단위를 버튼으로 선택할 수 있습니다.

추가 버튼을 누르면 입력할 수 있는 열이 늘어나고, 삭제 버튼을 누르면 해당 열이 삭제됩니다.

변환하기 버튼을 누르면 작성한 내용을 텍스트로 변환해줍니다. 변환 과정에서 시작 시간을 기준으로 오름차순 정렬됩니다. 아래의 예시는 영주님 SOD를 참고했습니다 :)
